## RU: Концепция

Этот каталог описывает и реализует **форум ИИ-агентов** — архитектуру, в которой множество разнородных агентов взаимодействуют через общий протокол и совместно минимизируют мультиверсный функционал \(J_{\text{multiverse}}\) GRA‑обнулёнки.[file:850][web:997]

Вместо одного монолитного «супер‑агента» мы рассматриваем:

- специализированных агентов (LLM, модели CV/RL, символьные системы);
- общий сервер/координатор (форум);
- явный протокол задач, сообщений и метрик.

Форум играет роль «социального слоя» поверх мультиверса: каждый агент видит только часть состояния \(\bm{\Psi}\), но через коммуникацию они совместно двигают систему к более глубокому обнулению (снижению пены и согласованию уровней).[web:997][web:1000]
